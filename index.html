Projects

  • Camel: It is the improved version of the existing offline payments protocol I made. Camel uses 4-way handshakes using QR codes to create State Channels and stores them locally in indexedDB. GitHub repo
  • EVM-PMPC: I am still working on this project, its a protocol I am building from scratch to enable private multi-party computation on data in EVM chains. It can be used for private DeFi, private prediction markets and more. GitHub repo
  • Intent-Swaps: Built an intent-based swap system on a local Arbitrum fork where solvers discover and fulfill signed user intents. Used EIP-712 typed-data signatures for secure off-chain message signing and on-chain verification. GitHub repo
  • Rotor: An privacy focused payments protocol built on Stellar which uses commitment schemes for private transactions. Used Noir for writing the circuits and UltraHonk proving scheme for proving in a fast and effective way. GitHub repo
  • BOO: Bitcoin Optimistic Oracle, an Optimistic Oracle system built on Stellar to provide off-chain data, on-chain. Implemented an architecture to provide incentive to data providers with a slash-and-reward mechanism GitHub repo
  • Offline Payments: A system to perform offline payments using QR codes and setting up trustlines between users. Used alloy to implement the backend in Rust. GitHub repo
